# スキル一覧（決まった作業を覚えさせる場所）

**考え方**：一度やった「決まった作業」は手順書（スキル）にして残す。
次からは codex がそのスキル通りに実行するので、**数が増えるほど精度と速さが上がる**。

## いまあるスキル
| スキル | 用途 |
|--------|------|
| `add-question/` | 診断に設問を追加する |
| `add-result-type/` | 結果タイプ（スタイル）を追加・調整する |
| `share-card/` | 結果のシェア画像／OG表示を作る・直す |
| `_template/` | 新しいスキルを作る時の雛形 |

## スキルの増やし方（精度を上げるループ）
1. 決まった作業をやる
2. 終わったら `_template/SKILL.md` をコピーして新スキルを作る
3. 手順・入力・完了条件・注意点を書く
4. この表に1行足す
5. 次回から codex はそのスキル通りに実行 → ブレが減る

## 使い方（codex側）
- 作業前に該当スキルを探す → あればその手順で実行
- 無ければ実行後にスキル化を提案し、承認されたら追加する
