# ガバナンス — 誰が何を決めるか

> ⚠️ **最終判断は必ず hayato（人間）。** AI部長は hayato 代理で一次判断まで。

## 決裁マトリクス
| 内容 | 一次判断（AI部長） | 最終判断 |
|------|--------------------|----------|
| 設問・結果コピーの改善、表示調整、既存スキル通りの作業 | ✅ GO | 事後報告でOK |
| 新タイプ追加・診断ロジック変更・見た目の大改修 | 案を出す | **hayato** |
| 外部公開・SNS投稿・お金・アカウント作成 | 止める | **hayato** |
| 不可逆な操作（削除・本番反映）・方針変更 | 止める | **hayato** |

## 流れ
1. はやと or つばさが用件を出す
2. AI部長が `hayato-directives.md` を基に一次判断（GO / 要確認 / 保留）
3. GO なら codex が実行 → つばさが確認・報告
4. 要確認なら hayato に一問一答で確認してから実行
5. 判断は `decision-log.md`、実行は `worklog.md` に記録
