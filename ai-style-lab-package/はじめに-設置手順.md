# ai-style-lab に AIカンパニーを設置する手順

## 中身
- `AGENTS.md` … codex が自動で読む入口（**リポのルートに置く**）
- `ai-company/` … AIカンパニー一式（AI部長・スキル・ルール・司令塔）

## 置き方（どちらか）

### A. codex に任せる（かんたん）
Cursor内のcodexに、このフォルダを渡してこう頼む：
```
このzipの中身を ai-style-lab のルートにそのまま展開して。
AGENTS.md はルート直下、ai-company/ はルート直下のフォルダとして置いて。
既存ファイルは上書きしない。置いたら中身を読んで、これからこの体制で動いて。
```

### B. 手で置く
1. `AGENTS.md` を ai-style-lab のルートにコピー
2. `ai-company/` フォルダごと ai-style-lab のルートにコピー
3. Cursorで開き直す（codexがAGENTS.mdを読み込む）

## 置いたあとの動かし方
1. はやとが「◯◯やって」と言う
2. codex は `AGENTS.md` → `ai-company/ai-bucho/hayato-directives.md` → 該当スキル の順で読む
3. 実行 → `ai-company/docs/worklog.md` に記録 → 報告

## AI部長を育てる
新しい方針を出したら、`ai-company/ai-bucho/hayato-directives.md` の一番下に1行足す。
codex は毎回そこを読むので、貯めるほど「はやとの意思」に沿って動く。

## スキルを増やす
決まった作業をやったら、`ai-company/skills/_template/SKILL.md` をコピーして新スキルに。
増やすほど精度と速さが上がる。
