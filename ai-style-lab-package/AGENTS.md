# AGENTS.md — codex への入口（ai-style-lab）

このリポジトリ（ai-style-lab）は **AIカンパニー体制**で運営します。
codex は作業を始める前に、必ず次を読んでください。

## まず読む（順番）
1. `ai-company/ai-bucho/hayato-directives.md` — **はやとさんの方針・指示の蓄積（最優先で従う）**
2. `ai-company/docs/operating-rules.md` — 全社共通ルール
3. `ai-company/docs/governance.md` — 誰が何を決めるか（最終判断は必ず hayato）
4. `ai-company/company/ai-style-lab.md` — この会社（AIスタイル診断社）の役割
5. `ai-company/skills/` — 決まった作業の手順書（該当があれば必ずスキル通りに実行）

## 動き方（要点）
- **はやとさんの指示 → AI部長が一次判断 → codex が実行 → 結果を報告**
- 決まった作業は必ず `ai-company/skills/` の該当スキルに沿う。無ければ実行後にスキル化を提案する
- 実行したら `ai-company/docs/worklog.md` に1行追記する

## 絶対ルール（違反しない）
- **プロジェクトをまたいだ編集をしない**（ai-style-lab の外を触らない）
- **ネガティブ表現を使わない**（弱点/向いていない/劣る 等）
- **外部公開・お金・不可逆な操作・最終決定は、必ず hayato の承認を取る**
- 迷ったら勝手に進めず、一問ずつ確認する
