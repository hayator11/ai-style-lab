# 結果タイプ（スタイル）を追加・調整する

## いつ使う
新しいスタイルを足す、または既存タイプの説明・持ち味コピーを直す時。

## 入力
- タイプ名（キャッチcoピー）
- 持ち味（3つほど）
- 一言説明
- どんな設問スコアでこのタイプになるか（判定条件）

## 手順
1. 既存タイプ定義ファイルの書式を確認する
2. 同じ書式でタイプを追加（キー・名前・説明・持ち味）
3. 判定ロジック（スコア→タイプ）に条件を追加する
4. どのスコアでも必ず1タイプに着地するか確認（取りこぼしを作らない）
5. 診断を数パターン通して、表示・コピーを確認する

## 完了条件（DoD）
- 新タイプが判定され、結果画面に持ち味・説明が前向きに表示される

## 注意
- 優劣で書かない。すべて「持ち味」として対等に書く
- ネガティブ表現を使わない
- 新タイプ追加は影響が大きいので、**公開前に hayato に確認**
